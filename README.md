# StayNest 🏡

StayNest is a full-stack Airbnb-inspired web application where users can:
- Create and manage property listings
- Upload images using Cloudinary
- Search listings by location and category
- Leave reviews and ratings
- Authenticate securely (Login / Signup)
- delete review and listing
- update listing

## Tech Stack
- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap / Tailwind CSS
- Cloudinary
